# Portfolio Site

An aesthetic, premium portfolio website built with React, Vite, and Framer Motion.

## Features
- **Modern Design**: Glassmorphism, smooth gradients, and dark mode aesthetics.
- **Animations**: Powered by `framer-motion` for scroll reveals and interactive elements.
- **Responsive**: Fully responsive layout using CSS Grid and Flexbox.
- **Performance**: Optimized with Vite.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Project Structure
- `src/components`: Reusable UI components (Hero, About, Projects, Contact).
- `src/index.css`: Global styles and design system variables.
- `src/App.jsx`: Main application layout.

## Customization
- Update `src/components/Projects.jsx` to add your own projects.
- Replace `src/assets/profile.png` with your own photo.
- Modify colors in `src/index.css` variables.
